# GW_GRB_Joint_Simulation

Simulated joint analysis of gravitational-wave (GW) and gamma-ray burst (GRB) events using coincidence statistics.

## Project Overview
This project simulates GW and GRB trigger populations and performs a basic joint analysis using temporal and spatial coincidence tests. The goal is to build a conceptual understanding of multimessenger pipelines.

## Objectives
- Simulate GW and GRB trigger populations
- Implement temporal and spatial coincidence tests
- Compute joint ranking statistics and simple summary outputs
- Build conceptual understanding of multimessenger pipelines

## At a Glance
- Inputs: simulated GW/GRB trigger CSVs with time and sky position
- Outputs: coincidence table, summary stats, and sky/analysis plots
- Config: thresholds and priors live in `src/config.py`
- Data flow:
```text
simulate_gw.py + simulate_grb.py
  -> coincidence.py
  -> statistics.py
  -> plotting.py + analysis_plots.py
```

## Setup (Windows with WSL2 - required)

### Why WSL2?
This project uses `healpy` for HEALPix spherical data analysis. On Windows, `pip install healpy` fails because:
- Healpy has no pre-built Windows wheels on PyPI
- It requires C++ compilation with external libraries (cfitsio, healpix_cxx, sharp)

Solution: use WSL2 (Windows Subsystem for Linux) where healpy installs normally.

### Step 1: Install WSL2 (one-time setup)

Option A: automatic (recommended)
1. Open PowerShell as Administrator
2. Run: `wsl.exe --install Ubuntu`
3. Restart when prompted
4. Ubuntu will finish installing automatically
5. Create a WSL username and password

Option B: manual (if automatic fails)
1. Open PowerShell as Administrator
2. Run: `wsl.exe --install --distribution Ubuntu`
3. Restart
4. Follow on-screen setup

Verify installation:
```powershell
wsl --list --verbose
```
You should see Ubuntu with version 2.

### Step 2: Create project environment (one-time setup)

Recommended: run the setup script

## Analysis Plots
The analysis plots summarize coincidence timing, sky separation, and ranking:
- `figures/analysis_delta_t_hist.png`: histogram of time offsets (Delta t) for coincident pairs
- `figures/analysis_angle_hist.png`: histogram of angular separation (theta) for coincident pairs
- `figures/analysis_rank_lambda_hist.png`: histogram of log10 Lambda ranking statistic
- `figures/analysis_pH1_hist.png`: histogram of posterior signal probability p(H1 | D)
- `figures/analysis_dt_vs_angle.png`: scatter of Delta t versus angular separation
- `figures/analysis_no_coincidences.png`: placeholder if no coincidences exist

These are generated by `src/analysis_plots.py` and included in `run_pipeline.sh`.
