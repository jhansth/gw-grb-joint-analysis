# Simulated Triggers

This folder contains the simulated trigger catalogs.
They are regenerated by `src/simulate_gw.py`, `src/simulate_grb.py`, or `src/pipeline.py`.

Files:
- `gw_triggers.csv`
- `grb_triggers.csv`

Notes:
- `grb_triggers.csv` may include correlation diagnostics such as
  `is_correlated`, `parent_gw_id`, `time_offset_s`, and `sky_offset_rad`.
